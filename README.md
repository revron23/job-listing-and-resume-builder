# Job Listing and Resume Builder

This repository contains an academic project focused on building a web-based job listing and resume builder application.

## Project Overview
The application allows users to browse job listings and create structured resumes through a simple and interactive interface. The project was developed to understand full-stack application flow and UI-to-backend integration.

## Features
- Job listing interface
- Resume builder workflow
- Web-based user interface
- Resume preview and generation
